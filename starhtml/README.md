# StarHTML Kanban Board

Einfaches Kanban-Board implementiert mit [StarHTML](https://starhtml.com/).

## Features

- Drei Spalten: Todo, In Progress, Done
- Tasks hinzufügen
- Tasks zwischen Spalten verschieben
- Reaktive Updates ohne Page Reload

## Installation

```bash
pip install -r requirements.txt
```

## Starten

```bash
python app.py
```

Die App läuft auf http://localhost:5001
